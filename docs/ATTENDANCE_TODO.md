# Attendance Module TODO Roadmap

This roadmap captures future enhancements for a full-fledged attendance system.

## Platform and UX

- [ ] Realtime attendance board for teacher, admin, and parent views
- [ ] Bulk mark templates by class schedule and day pattern
- [ ] Fast keyboard mode for large class attendance entry
- [ ] Mobile-first teacher check-in flow for low-bandwidth usage

## Compliance and Verification

- [ ] Geofenced attendance marking for on-campus verification
- [ ] Optional QR check-in mode for supervised attendance capture
- [ ] Optional biometric integration points for hardware devices
- [ ] Attendance change audit timeline with actor and timestamp history

## Parent and Student Communication

- [ ] Parent instant notifications for absence and late marks
- [ ] Daily attendance digest via SMS, WhatsApp, and email
- [ ] Student self-view attendance timeline with explanations
- [ ] Excuse note submission and approval workflow

## Analytics and Intervention

- [ ] Chronic absenteeism risk scoring by configurable thresholds
- [ ] Weekly and monthly attendance trend dashboards
- [ ] Class, grade, and school comparison heatmaps
- [ ] Intervention task generation for homeroom and counselors

## Operations and Integrations

- [ ] Public API endpoints for attendance ingest and sync
- [ ] CSV bulk import with duplicate detection and preview
- [ ] Payroll linkage for staff attendance and lateness summaries
- [ ] Calendar integration for school holidays and attendance exclusions

## Governance and Security

- [ ] Fine-grained role matrix for attendance actions by module area
- [ ] Approval flow for attendance edits after lock period
- [ ] Report export watermarking and policy-based redaction
- [ ] Data retention and archival controls by academic year
